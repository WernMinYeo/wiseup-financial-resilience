# WiseUp Financial Resilience System

WiseUp is an AI-driven behavioural finance ecosystem designed to help Malaysian students and early-career graduates build sustainable financial habits through automation, gamification, and intelligent financial guidance using GXBank services.

# 🧩 Core Modules

| Module | Description |
|---|---|
| Smart Save | AI-powered automated savings system |
| Money Missions | Gamified saving habits and streaks |
| Lock Pocket | Behavioural guardrail for impulsive spending |
| Money Coach AI | Personalized financial insights and AI chatbot |
